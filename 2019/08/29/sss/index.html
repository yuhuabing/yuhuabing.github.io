hexo + github pages个人博客搭建流程

1.安装hexo前的准备

安装nodejs

解决网速问题用cnpm代替npm :npm install -g cnpm –registry=https://registry.npm.taobao.org

2.安装hexo

cnpm install -g hexo-cli

3.本地处理博客

新建博客文件夹 mkdir mblog

进入文件夹cd mblog/

初始化hexo hexo init

启动博客：hexo s

建新文章： hexo n “xxx”

文章位置： source /_posts/

清理：hexo clean

生成：hexo g

4.通过github pages把博客放到远端

创建仓库

安装插件 cnpm install –save hexo-deployer-git

设置_config.yaml

​ type:git

​ repo:github地址

​ brance:master

部署到远端：hexo d

5.换主题

下载主题 git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia

修改config文件 themes : yilia

后续操作：hexo clean hexo g hexo s hexo d
